Purpose
=======

This parser module is for parsing messages according to the new syslog standard
:rfc:`5424`.

.. note::
   ``pmrfc5424`` is built into rsyslog and must not be loaded with
   ``module(load="pmrfc5424")``.

Configuration Parameters
========================

.. note::

   Parameter names are case-insensitive; camelCase is recommended for
   readability.

Parser Parameters
-----------------

There are no configuration parameters for this module currently.
