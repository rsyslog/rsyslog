## rsyslog experimental container for syslog ETL functionality

This image is currently experimental and specialized for Vespa-oriented
ETL pipelines.

## Env vars

- ENABLE_UDP (default: on)
- ENABLE_TCP (default: on)
- ENABLE_RELP (default: on)
- ENABLE_VESPA (default: on)
- VESPA_NAMESPACE
- VESPA_DOCTYPE
- VESPA_SERVER
- VESPA_PORT
